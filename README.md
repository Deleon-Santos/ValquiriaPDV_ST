
# Valquíria PDV Web

## Objetivo
Sistema PDV Web desenvolvido em Streamlit, seguindo arquitetura MVC,
com SQLAlchemy para persistência e geração de cupom em PDF.

## Funcionalidades
- Login de usuário
- Cadastro de produtos
- Carrinho de vendas
- Fechamento de venda
- Relatórios
- Geração de PDF

## Como usar
```bash
pip install -r requirements.txt
streamlit run app.py
```
